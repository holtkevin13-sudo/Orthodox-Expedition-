How to use this tool
  1. 1
    Click Generate Index below. It fetches all saint filenames from the Companion repository.
  2. 2
    Click Copy JSON to copy the result.
  3. 3
    In your GitHub Expedition repo, open games/saints-index.json and replace the entire contents with the copied text. Commit.
  4. 4
    Done — the Icon Gallery will load instantly using the updated index.
